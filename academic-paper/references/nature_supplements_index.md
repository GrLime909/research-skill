# Nature-Style Writing Supplements — Index

> **Purpose**: Quick-reference for four supplementary references adapted from nature-writing. These complement (not replace) the existing academic-paper references and templates. They add writing-level guidance for the IMRaD sections where the two skills overlap, plus journal-specific prose style adjustments.

## When to Use Each File

| Existing academic-paper reference | Supplementary file | What it adds |
|---|---|---|
| `paper_structure_patterns.md` §IMRaD Introduction | `nature_introduction_variants.md` | 9 introduction variants (pipeline/technical-challenge patterns) with sentence skeletons and "not recommended" patterns |
| `imrad_template.md` §Methodology | `nature_method_guide.md` | Three-element per-module pattern (motivation → design → advantage), pre-writing checklist, forbidden-vague-phrases table, clarity checks |
| `abstract_writing_guide.md` (structured 5-component) | `nature_abstract_title_guide.md` | Unstructured Nature-style abstract paragraph + diagnostics; title drafting rules with patterns and anti-patterns |
| `imrad_template.md` §Discussion → Conclusion | `nature_discussion_conclusion_guide.md` | Discussion-interpretation syntax, rival-address rules, "observe vs interpret" distinction; conclusion overclaim checklist, limitation-type guidance |
| `journal_submission_guide.md` (submission logistics) | `nature_journal_writing_style.md` | Writing-level adjustments per journal (Nature/Nat Comms/Generic) — audience, hedging, word budget tables, abstract treatment |

These supplements are useful in:

| Mode | Recommended supplements |

## Relationship to the IMRaD Template

The IMRaD template and these supplements cover **different writing layers**:

```
IMRaD template:                These supplements:
┌─────────────────────┐       ┌──────────────────────┐
│ Structural skeleton │  +   │ Writing-level choices │
│ What goes in each   │       │ How to phrase each    │
│ section/subsection  │       │ section (variants,    │
│                     │       │ sentence skeletons,   │
│                     │       │ diagnostics)          │
└─────────────────────┘       └──────────────────────┘
```

- **IMRaD template** tells you *what* to write (sections, subsections, content prompts)
- **These supplements** tell you *how* to write it (funnel patterns, module triads, hedge calibration, overclaim checks)
